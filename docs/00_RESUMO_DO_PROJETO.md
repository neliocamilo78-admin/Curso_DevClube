# Curso_DevClube - Workspace de Desenvolvimento

## Objetivo do projeto

Este projeto é usado para aprendizado de HTML, CSS, JavaScript, Git, GitHub e publicação com GitHub Pages.

## Estrutura atual

- index.html: página inicial
- projetos.html: página de projetos
- contato.html: página de contato
- galeria_de_fotos.html: galeria de imagens
- img/: pasta de imagens
- docs/: documentação do aprendizado

## Nível atual

Iniciante em GitHub, HTML, CSS e JavaScript.

## Fluxo de trabalho

1. Editar arquivos no VS Code.
2. Salvar com Ctrl + S.
3. Fazer commit.
4. Sincronizar com GitHub.
5. Pedir revisão ao ChatGPT.

## Como pedir ajuda ao ChatGPT

Sempre que precisar de revisão, enviar:

- print da estrutura do projeto;
- conteúdo do arquivo alterado;
- erro apresentado;
- objetivo da alteração.
